Exemplos práticos

A melhor maneira de aprender a Flexbox é usá-la para mover um monte de caixas ao redor. Então, é exatamente isso que vamos fazer no capítulo sobre Flexbox, como você pode ver. E isso não se aplica apenas à leiaute—todo o tutorial segue esta filosofia prática, desde Páginas Web Básicas até Tipografia da Web.

Nós desenhamos meticulosamente nossos projetos de exemplo para guiá-lo gentilmente através de um novo aspecto do HTML e CSS enquanto ainda demonstramos conceitos com cenários concretos.

Isso significa que você raramente vai ler mais do que um parágrafo ou dois sem escrever algum código e ver o que ele faz (assim você nunca vai se entediar). Além disso, você terá uma tonelada de prática trabalhando em um editor de texto e verificando seu código em um navegador, assim como os desenvolvedores web reais fazem.
